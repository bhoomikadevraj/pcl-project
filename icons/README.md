# Icons

Add your app icons here:
- `icon-192.png` - 192x192 pixels
- `icon-512.png` - 512x512 pixels
- `favicon.ico` - 32x32 or 16x16 pixels

You can create icons using:
- [Favicon.io](https://favicon.io/) - Generate from text/emoji
- [RealFaviconGenerator](https://realfavicongenerator.net/) - Comprehensive icon generator
- Design tools like Figma/Canva

Suggested icon: Use the hand/accessibility symbol in emerald green (#10b981)
