## How to Contribute

### Reporting Bugs
- Use the issue tracker
- Include browser, OS, and steps to reproduce
- Add screenshots if helpful

### Suggesting Features
- Open an issue with [Feature Request] in title
- Describe the feature and use case
- Explain why it would be valuable

### Code Contributions

1. **Fork the repository**
2. **Create a branch**: `git checkout -b feature/your-feature-name`
3. **Make changes**: Follow existing code style
4. **Test thoroughly**: Check all features still work
5. **Commit**: Use clear, descriptive messages
6. **Push**: `git push origin feature/your-feature-name`
7. **Pull Request**: Describe your changes

### Code Style
- Use 2 spaces for indentation
- Use meaningful variable names
- Add comments for complex logic
- Follow existing patterns in codebase

### Testing Checklist
- [ ] Works in Chrome/Edge
- [ ] Works on mobile
- [ ] Camera/mic permissions work
- [ ] Authentication works
- [ ] Custom phrases save correctly
- [ ] No console errors

## Development Setup

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Open http://localhost:5500
```
