# Contributing

Thanks for helping improve Sign Sync.

## Before you start

- Keep changes focused and small.
- Preserve the static GitHub Pages setup.
- Avoid introducing new runtime dependencies unless they are clearly needed.

## Suggested workflow

1. Fork or branch the repository.
2. Make your change.
3. Verify the app still loads from `index.html`.
4. Submit a pull request with a short summary of the behavior change.

## Good contributions

- Bug fixes
- Accessibility improvements
- Small UI and content improvements
- Documentation updates
- Safe cleanup or refactoring

## Notes

- The app is deployed as a static site.
- Keep file paths and asset URLs compatible with GitHub Pages.
